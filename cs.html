CHAIN REACTION

BACKGROUND

This game was made as a project for CStuy 2018. The game was made in Processing, using Java. This game is a great demonstration of states.

DESCRIPTION

Chain Reaction is a 10 Level, single-player game that has the player aiming for a certain amount of points. In each level, the player is given one click to cause the largest chain reaction possible. The better and longer the chain reaction, the more points the player recieves! Failure will make the player redo the level.

HOW TO PLAY

Very simple! Mouse Clicks!

PONG (2 PLAYERS)

BACKGROUND

This game was made as a project for CStuy 2018. The game was made in Processing, using Java. This game is a great demonstration of short-circuiting and basic physics.

DESCRIPTION

This version of Pong is for 2 Players. Each player is aiming to get 7 points first. It's the good ol' Pong everyone knows!

HOW TO PLAY
Player 1: W+S
Player 2: UP+DOWN
Start: ENETER
